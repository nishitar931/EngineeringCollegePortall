# College Counseling Prediction Portal

An AI-powered, rank-based eligibility filtering application designed to help engineering aspirants predict their college admissions. This portal streamlines the counseling process by evaluating student entrance exam ranks against historical cutoff databases across multiple rounds.

## 🚀 Features

- **Rank-Based Filtering:** Accurate prediction algorithms matching student ranks with multi-year cutoff data.
- **Dynamic Database Integration:** Structured query handling using a lightweight SQLite backend.
- **Responsive Dashboard:** Clean UI built with HTML, CSS, and vanilla JavaScript for quick search filtering.
- **Multi-Round Support:** Handles cutoffs across different allocation rounds for a realistic prediction outcome.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Node.js / Python (Depending on your final implementation)
- **Database:** SQLite
- **AI/Framework Integrations:** Architecture scalable for LLM agent integration (LangChain)
